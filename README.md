# Clearance Control

Urgent same-day clearance and relocation service in Singapore.

## Deployment

This site is deployed on Vercel.

## Local Development

Simply open `index.html` in a browser.

## Update Instructions

1. Update WhatsApp number in `index.html` (search for `6512345678`)
2. Update Business Registration number (search for `202XXXXXX`)
3. Update availability window daily (line 26 in `index.html`)

## Optional: Add Image

Place your truck/license photo as `hero-image.jpg` in the root directory.
